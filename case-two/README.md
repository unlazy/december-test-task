### Original task
Given a string containing only lowercase English letters, remove duplicate letters so that 
every letter appears once and only once.

Ensure that your result is the smallest in lexicographical order among all possible results.

#### Example 1:
Input: `"bcabc"`

Output: `"abc"`

#### Example 2:
Input: `"cbacdcbc"`

Output: `"acdb"`

#### Example 3:
Input: `"cdadabcc"`

Output: `"adbc"`

#### Example 4:
Input: `"ecbacba"`

Output: `"eacb"`
